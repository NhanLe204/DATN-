# Pet-Heaven

Chào mừng bạn đến với **Pet-Heaven**! Đây là một dự án thú vị dành cho những người yêu thú cưng. Ứng dụng này có phần backend để quản lý dữ liệu và phần frontend để người dùng tương tác. Hãy cùng bắt đầu nào! 🐾

## Cài đặt dự án

Trước tiên, bạn cần cài đặt một số thứ cơ bản trên máy:

- Node.js (khuyến nghị phiên bản 16.x trở lên)
- npm (đi kèm với Node.js)
- Git (để clone dự án)

### Bước 1: Clone dự án

Clone dự án về máy của bạn và di chuyển vào thư mục dự án:

Hãy thay **https://github.com/ngocthanhnt00/pet-heaven.git** bằng link thật của dự án, và `<project-folder>` bằng tên thư mục dự án của bạn.

- git clone <clone nhánh cụ thể -b + (tên nhánh) > **https://github.com/ngocthanhnt00/pet-heaven.git** (**default**: main or master )

Khi cả backend và frontend đều chạy, bạn có thể truy cập ứng dụng qua trình duyệt ở `http://localhost:5173`. Nếu có lỗi, hãy kiểm tra lại terminal để xem thông báo nhé!

## Muốn đóng góp?

Nếu bạn muốn giúp dự án tốt hơn, hãy làm theo các bước sau:

1. Fork dự án trên GitHub.
2. Tạo một nhánh mới (`git checkout -b ten-nhanh-moi`).
3. Sửa đổi và commit code (`git commit -m "Thêm tính năng mới"`).
4. Push lên nhánh của bạn (`git push origin ten-nhanh-moi`).
5. Tạo một pull request để mình xem xét.
